About

An innovative crowd sourcing engine for mapping complex road geometry is developed. The project aims to aggregate the collection of geo-tagged data from crowd-sourcing agents (such as phones and vehicles) to output all tracks that belong to a road segment.

  • How we did it

    We mapped sensor data from crowd-sourcing agents (GPS data from cellphones) to segments from the OSM digital map. The extracted data is then re-purposed for various uses. Segment wise data enables different applications such as road geometry and road quality estimation. We then displayed the color coded aggregated statistics on a web application.
